# Weather App

A simple weather app UI which interacts with the free weather API service  (JS, HTML, CSS).

## Key Principles

- Asyncronous Javascript
- Event Loop (Call stack, NodeJS API, Callback Queue)
- Promises (.then(), .catch())
- async await
